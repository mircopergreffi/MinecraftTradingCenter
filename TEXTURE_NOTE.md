# Texture Note

The Trading Laptop item requires a texture file at:
`src/main/resources/assets/minecrafttradingcenter/textures/item/trading_laptop.png`

You can create a 16x16 pixel PNG image for the item texture. The mod will work without it, but will show a missing texture placeholder until you add one.

For a quick placeholder, you can copy any existing Minecraft item texture or create a simple colored square.
